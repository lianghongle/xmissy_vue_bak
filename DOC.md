1.安装node.js(就已经自带npm)
```
node -v
```
2.安装淘宝镜像
```
npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm -v
```
3.安装webpack
```
npm install webpack -g
webpack -v
```
4.安装vue-cli脚手架构建工具
```
npm install vue-cli -g
vue -V
```
5.用vue-cli来构建项目
```
vue init webpack project_name
```
6.启动
```
cd project_name
npm run dev
http://localhost:8080

```
7.安装element
```
npm i element-ui -S
```

8.Vuex
```
npm install vuex --save
```
